# **Общи задължителни изисквания**

Всеки проект трябва да съдържа:

## **1\. ООП структура**

* минимум **6 класа**:  
  * модел (данни);  
  * AI/ML логика;  
  * контролер/мениджър;  
  * събития/наблюдение;  
  * работа с файлове;  
  * работа с база данни.

## **2\. Комуникация между обекти**

* обекти да си взаимодействат;  
* използване на **Observer pattern (събития)**;  
* минимум 2 слушателя на едно събитие.

## **3\. Делегати/събития (Python стил)**

* callback функции;  
* event dispatcher;  
* custom event args (класове).

## **4\. LINQ еквивалент в Python**

* list comprehensions;  
* `map`, `filter`, `reduce`;  
* `sorted`, `groupby`;  
* анализ на данни.

## **5\. Файлове**

* запис на логове;  
* запис/четене на `.json`, `.csv`.

## **6\. Изключения**

* try/except;  
* custom exceptions;  
* обработка на грешки при AI модел.

## **7\. База данни**

* SQLite / MySQL;  
* минимум 3 таблици;  
* CRUD операции.

## **8\. AI компонент (ЗАДЪЛЖИТЕЛЕН)**

* проста ML логика:  
  * класификация;  
  * регресия;  
  * rule-based AI;  
  * или анализ на данни.

## **9\. Финална презентация**

* демонстрация на AI частта;  
* как работи моделът;  
* ограничения и грешки.
